# Roadmap

Future ideas to revisit after the timed round flow is working:

- Let players pick or switch teams before the game starts.
- Let players reclaim their existing seat from a different device if they leave or switch phones.
- Add prompt constraints/categories such as "An Animal", "A Celebrity", or "A celebrity doing something".
- Add a private admin analytics dashboard for the owner to review game usage: player counts, sessions, modes, durations, referral/source info, and other aggregate stats. Keep it hidden from normal players, likely behind a separate admin URL and authentication.
